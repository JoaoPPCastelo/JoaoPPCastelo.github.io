02

Moonshot is a webstore developed for the Web Application Development class at the University of Algarve, Portugal. The project consisted in the creation of an online store using the model-view-controller paradigm and using an opensource scripting language (PHP in my case), and/or a framework, where i used CodeIgniter. So, for this project i opted to create a webstore for photos/wallpapers, grouped by categories like animals, sports, people, etc, where the client, after sign in, can "buy" a selected photo and use it. The user also has access to all the photos that bought previously and a shopping cart. For the design, i created something simple and where the main focus are the photos available to the user. You can see and try the live website or check the source code available on GitHub.
